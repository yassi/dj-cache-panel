# Django Cache Panel

A Django Admin panel for browsing and inspecting Django cache backends defined in your `CACHES` setting.

![Django Cache Panel](https://raw.githubusercontent.com/yassi/dj-cache-panel/main/images/key_search_page_index.png)

## Overview

Django Cache Panel seamlessly integrates into your existing Django admin interface, providing a simple tool to view and manage your Django cache configuration without requiring any model definitions or database migrations.

## Key Features

- **View Cache Configuration**: See all cache backends defined in your `CACHES` setting
- **Django Admin Integration**: Seamless integration with Django admin styling
- **Permission Control**: Respects Django admin permissions and staff-only access
- **Multiple Cache Backends**: Support for viewing multiple cache configurations

## Requirements

- Python 3.9+
- Django 4.2+
